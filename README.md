# FoodCalTracker

앱인토스 (Vite + React + TDS) MyFitnessPal 사용자들이 한국 음식 데이터 부정확함에 지쳐있는 지금, 사진 한 장으로 한식 칼로리를 즉시 인식하고 식단 목표까지 연결해주는 한국 특화 영양 트래커 앱스토어 리뷰 기준 MyFitnessPal의 최대 불만은 '칼로리 부정확·음식 DB 부족'. 한국 음식(된장찌개, 비빔밥, 편의점 도시락 등)은 특히 데이터가 엉망이거나 없음. 직접 입력은 귀찮아서 3일 만에 그만두게 됨. 다이어트 앱을 깔았다 지웠다를 반복하는 핵심 이유가 '입력 불편함'.
