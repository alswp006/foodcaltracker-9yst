# FoodCalTracker

## Platform
앱인토스 (Vite + React + TDS)

## One-liner
MyFitnessPal 사용자들이 한국 음식 데이터 부정확함에 지쳐있는 지금, 사진 한 장으로 한식 칼로리를 즉시 인식하고 식단 목표까지 연결해주는 한국 특화 영양 트래커

## Problem
앱스토어 리뷰 기준 MyFitnessPal의 최대 불만은 '칼로리 부정확·음식 DB 부족'. 한국 음식(된장찌개, 비빔밥, 편의점 도시락 등)은 특히 데이터가 엉망이거나 없음. 직접 입력은 귀찮아서 3일 만에 그만두게 됨. 다이어트 앱을 깔았다 지웠다를 반복하는 핵심 이유가 '입력 불편함'.

## Target Users
다이어트·체중 관리 중인 20-40대 한국 여성, 헬스 식단 관리하는 PT 등록자·운동 유저, 편의점·배달음식 위주 식사하는 직장인

## Core Features
1. 음식 사진 촬영 → AI 자동 인식 및 칼로리·영양소 추정(한국 음식 특화 학습)
2. CU·GS25·세븐일레븐 편의점 제품 바코드 스캔 → 즉시 영양정보 표시
3. 식약처 공공 영양 DB + 크라우드소싱으로 한국 음식 데이터 보강
4. 일일 목표 칼로리 기반 잔여량 시각화(간단한 링 UI)
5. 배달의민족·쿠팡이츠 메뉴 연동(인기 메뉴 칼로리 사전 조회)

## Monetization
- Model: freemium_subscription
- Estimated MRR: $3,100
- Reasoning: 무료: 하루 3회 사진 인식 / 프리미엄 ₩6,900/월: 무제한 + 주간 리포트 + 배달앱 연동. 다이어트 앱 결제 의향 한국 시장에서 검증됨(눔 코리아 성공 사례). 건강 카테고리 과거 성과 미흡했으나, 해당 앱들은 수익화 모델이 없었음 - 명확한 프리미엄 구분으로 차별화. 프리미엄 200명 목표.
